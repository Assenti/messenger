# Messenger web application

##### Used stack:
- JS (ES6), 
- React.js,
- Redux (state manager),
- Socket.io (web sockets),
- SCSS (own styles)


###### App starting in development mode (default port: 3000)
```bash
npm start
```

###### App building for production
```bash
npm run build
```
